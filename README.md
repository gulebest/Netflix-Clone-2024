# Netflix-Clone

## About

This Netflix-clone is a responsive React web application that mimics the core features of the Netflix streaming platform. It includes a dynamic banner highlighting featured content, multiple categorized rows displaying movies and TV shows, and reusable components like Header, Footer, Banner, and RowList. The app fetches data from an external API, showcasing modern React techniques such as hooks, component composition, and asynchronous data fetching for a seamless user experience.

## Features

- Responsive design for all screen sizes  
- Featured content banner with dynamic updates  
- Multiple rows categorized by genres or types  
- Modular, reusable React components  
- API integration for real-time movie and show data  
- Clean and intuitive UI

## Technologies Used

- React (Functional Components & Hooks)  
- JavaScript (ES6+)  
- CSS Modules  
- External Movie API (e.g., TMDB)  
- React Router (optional)
